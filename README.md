# Randomizer for PMDO/PMDC/RogueEssence
This project aims to allow extreme flexibility in randomizing everything in PMDO, from monster types and health, to item prices and effects, to dungeon tiles and floors.

## Setup
This mod requires one external dependency, SharpYaml, for saving and loading presets.
To set it up properly:
- Click download package on https://www.nuget.org/packages/SharpYaml/2.1.4
- Change the extension to `.zip` or open in any archive program
- Extract `lib/netstandard2.0/SharpYaml.dll` into the `Libraries` folder.
